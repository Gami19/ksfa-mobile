# KSFA Mobile デモ準備ガイド

## Phase 4: 接続・デモ準備

### 1. ローカルサーバー起動（外部アクセス許可）

```bash
npm run dev
```

このコマンドは `-H 0.0.0.0` オプションが含まれており、外部からのアクセスが可能です。

**ローカルホストのみで起動する場合:**
```bash
npm run dev:local
```

### 2. PCのIPアドレスを確認

#### macOS / Linux
```bash
ifconfig | grep "inet " | grep -v 127.0.0.1
```

#### Windows
```cmd
ipconfig
```

例: `192.168.1.100` のようなIPアドレスを確認します。

### 3. USBテザリング接続

1. スマホとPCをUSBケーブルで接続
2. スマホで「USBテザリング」または「インターネット共有」を有効化
3. スマホのブラウザから `http://[PCのIP]:3000` にアクセス
   - 例: `http://192.168.1.100:3000`

### 4. ホーム画面に追加（PWAインストール）

#### iOS (Safari)
1. Safariでアプリを開く
2. 共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. ホーム画面にアイコンが追加されます

#### Android (Chrome)
1. Chromeでアプリを開く
2. メニュー（⋮）をタップ
3. 「ホーム画面に追加」または「アプリをインストール」を選択
4. ホーム画面にアイコンが追加されます

### 5. PWAアイコンの準備（オプション）

現在、manifest.jsonにはアイコンパスが設定されていますが、実際のアイコンファイルは未作成です。

アイコンを追加する場合:
1. `public/icon-192.png` (192x192px)
2. `public/icon-512.png` (512x512px)

これらのファイルを作成し、赤色（#FF6666）を基調としたデザインで作成してください。

### トラブルシューティング

#### 接続できない場合
- PCとスマホが同じネットワークに接続されているか確認
- ファイアウォールでポート3000がブロックされていないか確認
- `npm run dev` が正常に起動しているか確認（`-H 0.0.0.0` オプションが含まれているか）

#### PWAがインストールできない場合
- HTTPSまたはlocalhostでアクセスしているか確認（PWAはHTTPS必須）
- manifest.jsonが正しく読み込まれているか確認（ブラウザの開発者ツールで確認）
